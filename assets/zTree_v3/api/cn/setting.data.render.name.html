Function(name, treeNode)setting.data.render.name

概述[ 依赖 jquery.ztree.core 核心 js ]

用于使用自定义的方法动态渲染节点名称

默认值：null

v3.5.45+

Function 参数说明

nameString

对应 zTree 利用 setting.key.name 获取到的节点名称

treeNodeJSON

待渲染节点的 JSON 数据对象

针对根进行渲染时，treeNode = null

返回值Boolean

返回值是 渲染后的节点名称

setting & function 举例


	var setting = {
		data: {
			render: function(name,treeNode){
				return name + treeNode.id;
			}
		}
	};
